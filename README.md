# Weather App

**Weather App** — це навчальний десктопний застосунок для перегляду поточної погоди та прогнозу, розроблений на Python. Додаток отримує дані про погоду через стороннє API, обробляє їх та відображає у зручному графічному інтерфейсі. Проєкт демонструє роботу з API, JSON-файлами та створення GUI-застосунків.

## Інтерфейс
Інтерфейс реалізований у вигляді графічного вікна з відображенням:

- поточної температури;
- опису погодних умов;
- іконки погоди;
- прогнозу на кілька днів;
- вибору міста.

Для відображення погодних умов використовуються графічні іконки відповідно до стану погоди.

---

## Використані технології
- **Python**
- **Tkinter** — створення графічного інтерфейсу
- **Requests** — робота з HTTP-запитами
- **OpenWeather API** — джерело погодних даних
- **JSON** — збереження та обробка даних
- **python-dotenv** — робота з .env файлом

## Установка та запуск

1. Клонування репозиторію:

git clone https://github.com/your-username/Weather-app.git

cd Weather-app-main


2. Встановлення залежностей:

pip install requests python-dotenv

3. Створення `.env` файлу:

touch .env


4. Запуск проєкту:

python main.py
